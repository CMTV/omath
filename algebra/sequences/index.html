Понятие последовательности

Начем наше погружение в мир последовательностей с разбора самого этого понятия. Что это вообще такое?

Проведем аналогии с реальность жизнью. Например, вот последовательность действий для приготовления макарон:

  1. Заполнить кастрюлю водой
  2. Довести воду до кипения
  3. Засыпать макароны
  4. Готовить в течении 10 минут

Заметьте, что во всех примерах выше мы

О

Последовательность — функция f:N→X, которая каждому натуральному числу ставит в соответствие какой-то элемент множества X.

Числовая последовательность

Из введенного ранее определения последовательности следует, что мы нумеруем элементы какого-то произвольного множества X. Так вот, если множество X состоит только из чисел, то есть X⊆R, то выходит, что мы нумеруем числа. Поэтому, такие последовательности называют числовыми.

О

Числовая последовательность — последовательность, значениями которой являются числа.

Арифметическая прогрессия

Взгляните на последовательности ниже. Можете ли вы установить закономерность, по которой получены элементы каждой из них?

1, 2, 3, 4, 5, …6, 3, 0, −3, −6, …

В первом случае все очевидно. Это последовательность натуральных чисел. Что более важно для нас — каждый член этой последовательности получается прибавлением 1 к предыдущему члену. При этом, самый первый член этой последовательности равен 1. Говоря более формально, последовательность натруальных чисел можно описать вот так:

an​=an−1​+1a1​=1

Аналогичные рассуждения можно провести и для последовательности справа. Замечаем, что каждый следующий ее член получаем при вычитании 3 (или прибавлении −3) из предыдущего члена. Первый член равен 6. Формально эту последовательность можно описать так:

an​=an−1​+(−3)a1​=6

Заметьте, что обе эти последовательности похожи друг на друга в том смысле, что для получения следующего члена мы всегда прибавляем одно и то же число к предыдущему. Все подобные последовательности называются арифметическими прогрессиями.

О

Арифметическая прогрессия — числовая последовательность, каждый член которой (кроме первого) получается прибавлением к предыдущему константы, называемой разностью прогрессии.

Т
Общий член арифметической прогрессии

Член ai+Δ​, «сдвинутый» относительно члена ai​ на Δ «номеров» вправо или влево, отличается от него на число dΔ:

ai+Δ​=ai​+dΔ(i,i+Δ∈N, Δ∈Z)

Следствия:

  • Общий член через произвольный: am​=an​+d(m−n)
  • Общий член через первый член: an​=a1​+d(n−1)
Доказательство

Если Δ=0, то получаем очевидно выполняющееся равенство:

ai+0​=ai​+d⋅0=ai​

Докажем теперь справедливость формулы для натурального Δ с помощью метода математической индукции.

База индукции: при Δ=1 получаем

ai+1​=ai​+d

Это равенство выполняется по определению арифметической прогрессии. База индукции доказана.

Индукционный переход: пусть доказываемое равенство выполняется при Δ=k:

ai+k​=ai​+dk

По определению арифметической прогрессии следующий член ai+(k+1)​ можно получить, прибавив разность к предыдущему члену ai+k​:

ai+(k+1)​=ai+k​+d

Заменим ai+k​ на формулу выше и получаем следующее равенство:

ai+(k+1)​=ai​+dk+d=ai​+d(k+1)

Мы доказали, что равенство выполняется и для Δ=k+1. То есть, формула работает для любых натуральных Δ.

Доказательство для отрицательных целых Δ проводится так же по методу математической индукции по Δ′=−Δ.

■

Т
Характеристическое свойство арифметической прогрессии
Если {an​} — арифметическая прогрессия, то для любого n выполняется равенство

an​=2an−Δ​+an+Δ​​

Обратное тоже верно. Если равенство выше выполняется для любого n, то {an​} — арифметическая прогрессия.
